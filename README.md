# DSA Problem Tracker

A clean, responsive, terminal-themed web application designed to help developers track their Data Structures and Algorithms progress. It features local user authentication, comprehensive statistics tracking, and data portability.

## Features
- **User Authentication:** Secure local-only login/signup powered by browser `localStorage`.
- **Problem Management:** Add, edit, or delete problems with metadata including topics, difficulty, and links.
- **Analytics Dashboard:** Real-time stats, difficulty distribution bars, and a 49-day contribution heatmap.
- **Data Portability:** Export your progress to CSV with one click.
- **Responsive Design:** Fully mobile-optimized, terminal-inspired dark mode UI.

## Technology Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+).
- **Storage:** Browser `localStorage` (no backend required).

## Files
- `index.html`: Application layout and UI components.
- `style.css`: Custom theme styling with terminal aesthetics.
- `app.js`: Core logic, state management, and event handling.

## Deployment
This project is deployed for free using GitHub Pages. 
1. Navigate to the repository settings.
2. Enable "Pages" in the sidebar.
3. Your app will be live at the provided GitHub Pages URL.
